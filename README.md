# 📚 Estudos C#

Repositório dedicado aos meus estudos de **C#** e **.NET**, contendo exercícios, desafios e pequenos projetos desenvolvidos durante minha jornada de aprendizado.

## 🎯 Objetivo

Praticar os fundamentos da linguagem C# e evoluir gradualmente para o desenvolvimento back-end com .NET.

## 📖 Conteúdos

* Variáveis e tipos de dados
* Operadores aritméticos e lógicos
* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`, `do while`)
* Métodos e funções
* Arrays e listas
* Tratamento de exceções
* Programação Orientada a Objetos (POO)

## 🛠️ Tecnologias

* C#
* .NET
* Git
* GitHub
* Visual Studio Code

## 📂 Estrutura

Este repositório reúne exercícios e projetos criados durante os estudos da linguagem C#.

Cada pasta representa um tema, desafio ou projeto desenvolvido para praticar conceitos específicos.

## 🚀 Como executar

```bash
git clone https://github.com/arthurhenriquemelo2005/Estudos-C-.git
```

```bash
cd Estudos-C-
```

```bash
dotnet run
```
